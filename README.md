# Shashi Prabha | Power Electronics Engineer | IIT Dharwad

## Description

This is the personal website of Shashi Prabha, an Electrical Engineering graduate from IIT Dharwad, currently working as a power electronics hardware engineer at an electric vehicle startup, specializing in battery management systems (BMS) and hardware circuitry for bikes.

## Technologies Used

- **Frontend**: React 19.1.0
- **Build Tool**: Webpack 5
- **Styling**: Tailwind CSS 4.1.7
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

## Setup Instructions

### Prerequisites
- Node.js (recommended version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sprabha5/sprabha5.github.io.git
   cd sprabha5.github.io
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

### Development

3. **Start the development server**
   ```bash
   npm start
   ```
   This will start the webpack dev server and automatically open the website in your browser.

### Available Scripts

- **`npm start`** - Starts the development server with webpack-dev-server
- **`npm run build`** - Creates a production build in the `build` directory
- **`npm run format`** - Formats code using Prettier
- **`npm run deploy`** - Deploys the built site to GitHub Pages
- **`npm test`** - Runs tests (currently not configured)

### Building for Production

4. **Create a production build**
   ```bash
   npm run build
   ```
   The build files will be generated in the `build` directory.

### Deployment

5. **Deploy to GitHub Pages**
   ```bash
   npm run deploy
   ```
   This will build the project and deploy it to GitHub Pages.
